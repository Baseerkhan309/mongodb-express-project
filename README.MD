# MongoDB Express Project

A web application built using MongoDB, Express.js, EJS, and Mongoose.

## Features
- Create data
- Read data
- Update data
- Delete data
- MongoDB integration with Mongoose
- EJS templating
- Express routing
- Responsive styling

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- CSS
